เริ่มต้น

วิธีเริ่มใช้งาน

แนะนำการติดตั้งและนำระบบการออกแบบไปใช้กับโปรเจกต์ของคุณ

1. ติดตั้งโทเคน

คัดลอกไฟล์ colors_and_type.css และโฟลเดอร์ fonts/ ไปยังโปรเจกต์ของคุณ จากนั้น import ที่รากของแอป

/* app.css หรือ globals.css */
@import "./tokens/colors_and_type.css";

2. ใช้ตัวแปร CSS

.card {
  background: var(--canvas-white);
  border: 1px solid var(--stroke-neutral-lighter);
  border-radius: var(--radius-xl);
  padding: var(--spacing-xl);
  box-shadow: var(--shadow-custom);
}

3. ใช้ JSON tokens (สำหรับ Figma / Tailwind / iOS / Android)

ไฟล์ tokens.json สามารถแปลงผ่าน Style Dictionary หรือปลั๊กอิน Figma Tokens ได้โดยตรง

ขั้นตอนการตั้งค่า Style Dictionary พร้อมตัวอย่างไฟล์ config

4. React components (เฉพาะ web)

ดูไฟล์ต้นฉบับใน หน้าดาวน์โหลด — เป็น reference implementation ด้วย React + JSX ไม่ผูกกับ framework ใด

แนวทางการนำ components ไปใช้ใน Next.js / Vite / Remix
