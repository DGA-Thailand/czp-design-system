CZP ใช้ lucide-react เป็นไลบรารีหลัก (stroke 2px, rounded caps/joins) และ Material Symbols เป็นชุดเสริม — ห้ามใช้อีโมจิในส่วนของผลิตภัณฑ์

หลักการ: ไอคอนสื่อความหมายควรมีข้อความประกอบ · ขนาดมาตรฐาน 16/20/24 · สีตามบริบท (neutral/primary/status)

ตัวอย่างไอคอน

แนวทางการใช้

  • ขนาดเริ่มต้น 20px สำหรับ UI ทั่วไป, 16px ในป้าย, 24px ในปุ่มใหญ่
  • สีสืบทอดจาก currentColor — เปลี่ยนผ่าน color ของอีลีเมนต์ครอบ
  • ต้องมี aria-hidden="true" หากมีข้อความประกอบอยู่แล้ว หรือ aria-label หากใช้เดี่ยว
