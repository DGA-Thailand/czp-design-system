CZP Design System

ระบบการออกแบบสำหรับพอร์ทัลประชาชน (Citizen Portal) — ครอบคลุมสี ตัวอักษร ระยะห่าง และคอมโพเนนต์ที่ใช้ร่วมกันระหว่างเว็บและแอปพลิเคชันมือถือ เพื่อให้ประสบการณ์การใช้บริการของประชาชนสอดคล้อง ปลอดภัย และเข้าถึงได้อย่างเท่าเทียม

อธิบายวิสัยทัศน์ของระบบการออกแบบ หลักการสำคัญ และกลุ่มผู้ใช้งานเป้าหมาย (ประชาชนทั่วไป เจ้าหน้าที่ ผู้สูงอายุ ผู้พิการ)
