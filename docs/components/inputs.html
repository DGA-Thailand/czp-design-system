Inputs

ช่องกรอกข้อมูลขนาดมาตรฐาน 48px, มุมโค้ง 12px, รองรับสถานะ default / focused / error / disabled

สถานะ

รหัสผ่านสั้นเกินไป

แนวทาง

  • ทุก input ต้องมี <label> ที่มองเห็น ไม่ใช้ placeholder เป็น label
  • ข้อความ error แสดงด้านล่าง label และ input จะเปลี่ยนเป็นสี danger
  • ใช้ focused outline สีเขียวอ่อน (4px ring) สำหรับผู้ใช้คีย์บอร์ด

โค้ดตัวอย่าง

<Input label="อีเมล" value={v} onChange={setV} />
