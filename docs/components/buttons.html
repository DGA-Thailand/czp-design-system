Buttons

ปุ่มของ CZP เป็นทรง แคปซูล (border-radius: 1000px) สูงอย่างน้อย 48px เพื่อให้แตะได้สะดวกบนมือถือ และสื่ออารมณ์ที่เป็นมิตร มั่นคง

ชนิด

แนวทาง

  • ใช้ default เฉพาะ primary action หลักของหน้าจอ หลีกเลี่ยงมี default สองปุ่มในพื้นที่เดียวกัน
  • ใช้ destructive เฉพาะการกระทำที่ย้อนกลับไม่ได้ (ลบ ยกเลิกถาวร) และควรมี modal ยืนยันเสมอ
  • ใช้ outline คู่กับ default เพื่อให้ตัวเลือกที่สอง (เช่น "ยกเลิก") ไม่ดึงความสนใจเกินไป
  • ข้อความบนปุ่มควรเป็นคำกริยาสั้น ๆ เช่น "ยืนยัน", "บันทึก", "ดำเนินการต่อ" ไม่ใช่ "OK" หรือ "คลิกที่นี่"
